### System requirements:

## Getting started

#### Create virtual environment
```python3 -m venv env```

#### Activate the environment
```source env/bin/activate```

#### Clonet the Project

```cd <project_root>```

#### For migrations

Migrate Cmmand
```python manage.py makemigrations``` Creating migrations
```python manage.py migrate```

#### Run server
```python manage.py runserver```

#### Swagger URL
```localhost:8000/swagger/```
